# hub this is repository to store an e-commerce hub

- This project does not have a test project included with it.
- Which means no mock, TDD, Integration testing, etc

## everything about this small e-commerce is contained here

- This project is using docker, codespaces, android mobile(no laptop, desktop or any workstation except pure mobile and internet).

# List of Technologies used

1. .NET core
2. Docker (Containers)
3. git, github, codespaces
4. CSS, HTML5, JavaScript, C#, Shell, T-SQL, EntityFramework, etc
5. Android OS with Xiaomi HyperOS
6. AI tools - Gemini, claude, chatgpt (code was not generated by them, but i used them when i was stuck on a problem)
7. LinkedIn payed tutorials for understanding of codespaces
8. online courses.
